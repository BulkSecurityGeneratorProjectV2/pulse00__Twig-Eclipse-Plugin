Installation via Updatesite

The plugin can be installed via the Update Manager using the following updatesite:

http://pulse00.github.com/p2/

Requirements

Currently the plugin requires a development build of the PHP Development Tools (PDT). This build is available on the updatesite above. For details see this blog post. Eclipse >= 3.7 (aka Indigo) is recommended.

Due to the requirement to the PDT development build it is recommended to use a classic eclipse version. If you want to install the plugin into an existing eclipse build (Zend Studio, Aptana, FlashBuilder, etc), see the FAQ for instructions.
