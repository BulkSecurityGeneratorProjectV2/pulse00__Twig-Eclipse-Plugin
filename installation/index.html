Installation via Updatesite

The plugin can be installed via the Update Manager using the following updatesite:

http://p2-dev.pdt-extensions.org

Requirements

Eclipse >= 3.7.2 (aka Indigo) is required.

If you want to install the plugin into an existing eclipse build (Zend Studio, Aptana, FlashBuilder, etc), see the FAQ for instructions.
