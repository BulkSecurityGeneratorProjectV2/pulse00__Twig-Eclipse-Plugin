How can i install the plugin into an existing eclipse build?(Zend Studio, Aptana, FlashBuilder, [enter your build here])

The reason why the plugin currently needs a development build of PDT is because it contains new API methods for PDT extenders to index PHP sourcecode. This API is used by the plugin to parse and index Twig Extensions for tokenizers/functions/filters etc which is then used to implement codeassist in the editor. That's why with the current stable release of PDT twig-codeassist will not work - but syntax highlighting will.

So if your Eclipse build comes with a pre-installed version of PDT, you can probably use the syntax highlighting feature, but you'll not be able to use the codeassist features of the editor (although there might exist a workaround for codeassist.)

If your eclipse build is not bundled with PDT, you can simply install PDT + Twig through the Twig updatesite like in the vanilla eclipse installation.
