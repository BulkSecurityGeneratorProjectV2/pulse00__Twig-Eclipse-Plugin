How can i install the plugin into an existing eclipse build?(Zend Studio, Aptana, FlashBuilder, [enter your build here])

The Twig editor is based on the PHP Development Tools PDT. Therefore, any eclipse build which does not ship with built-in PDT needs to be configured to allow the twig plugin to resolve the PDT dependency.

So if your Eclipse build does not come with a pre-installed version of PDT, make sure that the default eclipse updatesite (currently Indigo) is available in your installation.
